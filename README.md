# Monty Hall Demo

Welcome to my Monty Hall simulation demo!

## Requirements ##
Java 11  
NodeJS  
Maven

#### Start backend ####
	cd backend  
	mvn install  
	mvn spring-boot:run  

#### Start client ####
	cd client  
	npm install  
	npm start  

##### Run all backend tests #####
	cd backend
	mvn test

##### Open cypress for e2e tests #####
	cd client
	mvn test
